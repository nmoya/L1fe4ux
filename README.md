# L1fe4ux

### Installation instructions
- `git clone http://github.com/nmoya/L1fe4ux`
- `npm install`
- Rename the file actions/keys.sample.js to action/keys.js
- Open the actions/keys.js file and insert your app keys
- `npm start`
